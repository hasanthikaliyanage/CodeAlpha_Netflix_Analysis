# 📊 CodeAlpha Netflix Data Analytics Project

## 🧾 Project Overview
This project is part of the Data Analytics Internship at CodeAlpha.  
It focuses on analyzing Netflix dataset using Python to extract insights, visualize trends, and perform sentiment analysis on content descriptions.

---

## 🎯 Objectives
- Perform Exploratory Data Analysis (EDA)
- Visualize data patterns and trends
- Analyze sentiment of Netflix descriptions
- Identify key insights from dataset

---

## 📁 Dataset
The dataset used: **Netflix Titles Dataset (netflix_titles.csv)**  
It contains information about movies and TV shows available on Netflix, including:
- Title
- Type (Movie / TV Show)
- Country
- Release Year
- Rating
- Description

---

## 🛠 Tools & Technologies
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- TextBlob (Sentiment Analysis)  
- Jupyter Notebook  

---

## 📊 Tasks Completed

### ✅ Task 2: Exploratory Data Analysis (EDA)
- Analyzed dataset structure
- Checked missing values
- Identified patterns in content type and release years
- Explored countries and categories

### 📈 Task 3: Data Visualization
- Movies vs TV Shows distribution
- Content trends over years
- Top countries producing content
- Rating distribution
- Genre analysis

### 😊 Task 4: Sentiment Analysis
- Analyzed Netflix descriptions
- Classified sentiment as Positive, Negative, Neutral
- Visualized sentiment distribution

---

## 🔍 Key Insights
- Netflix has more Movies than TV Shows
- Content production increased after 2015
- USA produces the highest number of titles
- Most descriptions are Neutral or Positive in sentiment
- Missing values exist in director and cast columns
